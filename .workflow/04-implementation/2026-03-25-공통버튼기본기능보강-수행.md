## 구현 메모
- 수정 파일: `src/app/components/common/Button.vue`
- 적용 내용:
  - `mode`를 `fill`, `ghost`, `outline`으로 고정하고 내부 계산 클래스로 베이스 스타일 정의
  - `type` 기본값을 `button`으로 지정
  - `loading`과 `disabled`를 하나의 비활성 상태로 계산
  - 텍스트/아이콘 버튼을 단일 템플릿 구조로 통합
  - `run` emit에 `MouseEvent` 전달
  - `inheritAttrs: false`와 `useAttrs()`를 사용해 네이티브 속성을 버튼에 전달
  - 아이콘을 선택값으로 낮추고 슬롯 또는 `label`로 텍스트 표시 가능하게 조정
- 판단 근거:
  - 공통 버튼의 기본 기능만 제공하고 시각 커스터마이즈는 호출부 `class`, `buttonClass`, `iconClass`로 열어두기 위함
  - 전역 CSS를 늘리지 않고 컴포넌트 내부에서 의미 기반 모드를 정리하는 것이 현재 범위에 적합함
