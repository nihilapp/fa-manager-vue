## 3. 구현 메모
- 에러 처리 유틸은 인증 리다이렉트, 로깅, 에러 콜백만 남기고 미사용 상태를 제거했습니다.
- `useGet`과 mutation 훅 전부를 객체 인자 기반으로 정리하고 `response`, `execute` 반환값을 맞췄습니다.
- mutation 훅은 각 파일에서 직접 `useMutation`을 호출하도록 바꾸고, 공통 응답 후처리만 `useApiRequest.ts`에서 공유했습니다.
