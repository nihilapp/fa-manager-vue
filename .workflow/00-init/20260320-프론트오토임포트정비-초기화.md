# 20260320-프론트오토임포트정비-초기화

## 1. 개요
프론트엔드 개발 생산성 극대화를 위해 Vue, Pinia, Vue Query 및 주요 프로젝트 유틸리티들에 대한 전역 오토임포트 설정을 수행한다.

## 2. 배경 및 목적
- **생산성 향상**: 반복적인 임포트 구문을 제거하여 비즈니스 로직 집중도 향상.
- **코드 간결화**: SFC(Single File Component) 내 스크립트 영역을 깨끗하게 유지.
- **표준화**: 프로젝트 전반에서 사용하는 유틸리티들을 동일한 이름으로 즉시 참조 가능하게 함.

## 3. 작업 범위
- `nuxt.config.ts` 내 `imports` 및 `presets` 설정 확장.
- Vue Query (`@tanstack/vue-query`) 관련 주요 훅 추가.
- 프로젝트 유틸리티 (`queryKeys`, `cn` 등) 추가.
- 기존 설정(lodash, luxon, uuid 등) 유지 및 보완.

## 4. 판단 근거
- 마스터의 "프론트에서 다른 무언가를 임포트 할 필요 없게 만드는 것"이 핵심 요구사항임.
- Nuxt 4의 오토임포트 기능을 최대한 활용하여 개발 경험(DX)을 최적화함.
