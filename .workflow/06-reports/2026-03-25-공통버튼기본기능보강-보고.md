### 주요 기능 변경
- `type` 기본값을 `button`으로 지정해 폼 내부 기본 submit 오동작을 방지했다.
- `disabled`와 `loading`을 하나의 비활성 상태로 계산해 중복 클릭을 막도록 정리했다.
- `run` 이벤트는 유지하면서 `MouseEvent`를 함께 전달하도록 확장했다.
- 아이콘 버튼과 텍스트 버튼을 하나의 구조로 통합하고, 슬롯 또는 `label`로 텍스트를 표시할 수 있게 만들었다.
- `class`, `buttonClass`, `iconClass`, 기타 네이티브 attrs를 버튼 엘리먼트에 반영할 수 있게 했다.

## 특이 사항
- 지침상 요구된 `context7 MCP`는 현재 세션에서 제공되지 않아 로컬 코드와 문서 기준으로 작업했다.
- 프로젝트 Tailwind 클래스 사용 범위가 제한될 수 있어 전역 CSS 확장 대신 컴포넌트 내부 클래스 계산 방식을 우선 적용했다.

- 필요 시 사이즈 체계(`sm`, `md`, `lg`) 또는 로딩 스피너를 별도 설계로 확장
