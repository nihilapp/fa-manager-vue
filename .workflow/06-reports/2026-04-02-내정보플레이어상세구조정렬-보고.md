## 작업 개요

## 결과 요약
### 주요 수정 파일
- `src/app/components/players/PlayerProfileDetail.vue`: 공통 상세 렌더링 컴포넌트 신규 추가
- `src/app/components/players/PlayerDetail.vue`: 공통 컴포넌트 래퍼로 단순화
- `src/app/pages/me/index.vue`: `mine` 계열 조회 실행과 공통 상세 렌더링 연결
- `src/app/pages/players/detail/[id].vue`: 누락된 캠페인/캐릭터 조회 보강 및 공통 상세 렌더링 연결

### 주요 변경점
- `me`와 플레이어 상세가 동일한 레이아웃과 섹션 구성을 공유한다.
- `me`는 `myInfo`, `myCharacterList`, `myCampaignList`, `mySessionList`를 그대로 사용한다.
- 일반 플레이어 상세는 `playerInfo`, `playerCampaignList`, `playerSessionList`와 `characters index + userId` 조회를 조합해 같은 뷰를 사용한다.
- 캐릭터/캠페인/세션 항목은 현재 존재하는 목록 페이지 경로로 이동 가능하게 정리했다.
