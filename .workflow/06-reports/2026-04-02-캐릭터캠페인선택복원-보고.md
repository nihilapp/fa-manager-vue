# 작업 보고 (Report)

## 작업 개요
- 작업명: 캐릭터캠페인선택복원
- 작업 목적: 캐릭터 에디터 버튼 문구를 실제 동작명으로 수정하고 캠페인 선택 드롭다운을 복원
- 진행 범위: 공용 캐릭터 에디터 폼 수정, 정적 검증, 워크플로우 문서 정리

## 결과 요약
- 생성 버튼이 `캐릭터 생성`, 수정 버튼이 `캐릭터 수정`으로 바뀌었다.
- 캠페인 목록을 드롭다운으로 선택할 수 있게 복원했다.
- submit payload에 `campaignId`를 다시 포함시켜 후속 API 분기 준비를 해뒀다.

## 주요 수정 파일
- `src/app/components/characters/CharacterEditorForm.vue`

## 검증 및 판단 근거
- ESLint 대상 파일 검증을 통과했다.
- 캠페인 선택값은 `null` 허용으로 구현해 미선택 상태도 표현할 수 있다.

## 후속 작업
- 실제 생성/수정 submit 시 `campaignId` 유무에 따른 API 분기 연결
