### 주요 기능 변경
- 상세보기 버튼을 공통 버튼으로 교체하고 일관된 아이콘/모드/호버 상태를 부여했다.
- 수정 버튼을 추가하되 관리자에게만 보이게 처리했다.
- 상태 컬럼을 텍스트에서 배지형 표현으로 바꿔 가독성을 높였다.
- 공통 테이블을 묵직한 회색 박스형에서 더 가벼운 카드형 목록 레이아웃으로 정리했다.
- 페이지네이션을 텍스트 나열형에서 조작 가능한 pill 버튼 그룹으로 개선했다.

## 특이 사항
- `context7 MCP`는 현재 세션에서 제공되지 않아 로컬 코드와 문서 기준으로 작업했다.
- 수정 버튼은 전용 수정 화면이 아직 없어 현재는 상세 페이지로 이동한다.

## 후속 작업
- 실제 플레이어 상세 페이지에 관리자 전용 수정 폼 또는 편집 모드 진입 경로 추가
- `DataTable.vue`를 사용하는 다른 목록 화면에서 새 스타일의 시각 적합성 확인
- 필요 시 컬럼 정렬, 필터, 선택 상태 등 에어테이블 계열 상호작용을 별도 작업으로 확장
