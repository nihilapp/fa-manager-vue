## 테스트 결과
### 단위 테스트
- 별도 단위 테스트는 없음

### 수동 테스트
- `pnpm.cmd run postinstall` 실행으로 `nuxt prepare`가 정상 완료되고 `.nuxt` 타입 생성이 성공했다.

### 통합 테스트
- `pnpm.cmd exec nuxi typecheck`를 실행했으나 기존 환경 문제로 실패했다.
- 실패 내용:
  - `tsconfig.json(4,27): error TS5103: Invalid value for '--ignoreDeprecations'`
  - `@vue/language-core` 모듈 누락으로 `vue-router/volar/sfc-route-blocks` 해석 실패
- 이번 작업 수정 파일 자체를 지목하는 타입 오류는 확인되지 않았다.

## 미결 사항
- 프로젝트 전역 타입체크는 기존 환경 이슈 해결 후 재실행이 필요하다.
- `currencyTransactions` 내부 중첩 관계(`user`, `character`)는 이번 범위에서 포함하지 않았다.
