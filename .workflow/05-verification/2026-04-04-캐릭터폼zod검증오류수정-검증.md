## 테스트 결과
### 단위 테스트
- 수행하지 않음. 해당 폼 스키마 전용 테스트가 현재 준비되어 있지 않다.

### 수동 테스트
- 정적 검토 기준으로 `invalid_union` 생성 경로를 제거했음을 확인했다.

### 통합 테스트
- 수행하지 않음. 별도 개발 서버 실행/브라우저 재현은 이번 턴에서 진행하지 않았다.

## 미결 사항
- 다른 도메인 폼에도 동일한 `z.union(...)` 패턴이 있으면 같은 문제가 재발할 수 있다.
- 패키지 버전 정합성 자체는 남아 있으므로 후속 점검이 바람직하다.
