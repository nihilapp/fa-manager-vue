  - `rg --glob "*.vue" "^\\s*(export\\s+)?(async\\s+)?function\\s+[A-Za-z0-9_]+\\s*\\(" src/app src/server`
  - 결과: 매치 없음
- 패턴 재검색:
  - `rg --glob "!*.vue" "^\\s*(export\\s+)?const\\s+[A-Za-z0-9_]+\\s*=\\s*(async\\s*)?\\([^)]*\\)\\s*=>|^\\s*(export\\s+)?const\\s+[A-Za-z0-9_]+\\s*=\\s*(async\\s*)?[A-Za-z0-9_]+\\s*=>" src/app src/server`
  - 결과: 매치 없음
- 타입 검증:
  - 실행 명령: `& .\\node_modules\\.bin\\tsc.cmd --noEmit`
  - 결과: 성공
- 실패 후 재시도 이력:
  - `npx tsc --noEmit`는 PowerShell 실행 정책으로 실패했고, 로컬 `tsc.cmd`로 대체했다.

## 미결 사항 및 위험도
- 현재 Git 워크트리에는 본 작업 외 사용자 변경 사항이 이미 존재한다.
- 본 작업은 대상 함수 선언 스타일만 수정했으며, 사용자 변경이 있던 다른 파일은 되돌리지 않았다.
- 동일 규칙의 재유입 방지는 아직 자동화되지 않았으므로, 후속으로 ESLint 규칙 또는 점검 스크립트 도입을 검토할 수 있다.
