# 설계 (Design)

## 요약
- 변경 대상: `CharacterEditorForm.vue`
- 설계 방향: 플레이어 등록 폼처럼 입력 필드와 버튼만 남기는 간결한 구조로 정리한다.
- 주요 결정 사항: 설명성 텍스트와 요약 박스는 제거하고, 필수 상태 메시지만 유지한다.

## 아키텍처 변화
### 수정 구성 요소
- `src/app/components/characters/CharacterEditorForm.vue`
  - 상단 안내 박스 제거
  - 섹션 제목 아래 설명 문구 제거
  - 제출 후 안내 박스 제거

## 로직 흐름
1. 로딩/에러 상태는 기존처럼 유지한다.
2. 정상 렌더링 시 바로 폼 필드와 버튼만 보여 준다.
3. 제출 시 추가 안내 없이 이벤트만 방출한다.

## 대안 검토
### 최종 선택
- 선택 이유: 마스터가 명시적으로 설명 없는 UI를 요구했다.
