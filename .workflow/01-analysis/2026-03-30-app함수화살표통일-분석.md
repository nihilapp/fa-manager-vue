# 분석 (Analysis)

## 요약
- 요청 내용: `src/app` 내부 함수 선언식을 화살표 함수로 통일하고, `player.store.ts`에 `playerInfo` 초기화 액션을 추가
- 범위: store 2개, player/query composable, util/useSetMeta 계열의 선언식 함수
- 구현 가능성: 가능

## 요구사항 요약
- `src/app` 내부에서 `function` 선언식 또는 내부 함수 선언식을 사용한 지점을 모두 화살표 함수로 바꾼다.
- 스토어 액션은 모두 화살표 함수로 통일한다.
- `player.store.ts`에 `playerInfo`를 `null`로 초기화하는 액션을 추가한다.
- 플레이어 상세 조회 오류 흐름에서도 새 초기화 액션을 사용할 수 있게 연결한다.

## 현재 상태
### 관련 파일
- `src/app/stores/player.store.ts`: 플레이어 목록/상세/내 정보 상태를 관리하는 Pinia store. 여러 액션이 함수 선언식이며 `playerInfo` 초기화 액션이 없음.
- `src/app/stores/common.store.ts`: 공통 내비게이션 상태를 계산하는 store. 현재 별도 액션 선언식은 없음.
- `src/app/composables/query/useGet.ts`: GET 요청 공통 composable. `execute`, `clear`, `refetch`가 내부 함수 선언식임.
- `src/app/composables/query/usePost.ts`, `usePut.ts`, `usePatch.ts`, `useDelete.ts`: mutation 성격 composable. `execute`, `clear`가 내부 함수 선언식임.
- `src/app/composables/query/useApiRequest.ts`: API fetch 옵션과 오류 처리 유틸. exported/local helper 일부가 함수 선언식임.
- `src/app/composables/useSetMeta.ts`: 메타데이터 설정 composable. helper와 exported composable이 함수 선언식임.
- `src/app/composables/players/*.ts`: 플레이어 관련 composable 5개. 모두 exported function 선언식이며 store 액션 호출을 포함함.
- `src/app/utils/api-error-handler.ts`: API 응답 에러 처리 유틸. helper 3개가 함수 선언식임.
- `src/app/utils/cn.ts`, `src/app/utils/icon-name.ts`: exported function 선언식 사용.

### 현재 로직 분석
- query composable 내부 함수들은 모두 선언 직후 반환 객체에서 참조되며, 호이스팅을 전제한 호출 순서는 없음.
- player composable은 store 액션을 직접 호출하므로, `playerInfo` 초기화 액션 추가 시 상세 조회 실패 흐름을 안전하게 정리할 수 있음.
- `useGetPlayerInfo.ts`는 에러 시 `playerStore.cl()`를 호출하고 있어 현재 코드 자체가 불완전함. 새 액션 추가와 함께 정상 store 액션 호출로 교정하는 것이 필요함.

## 영향도 평가
### 직접 영향
- 함수 선언식을 사용하는 `src/app` 파일들의 선언부 문법
- `player.store.ts`의 반환 액션 목록
- `useGetPlayerInfo.ts`의 오류 처리 흐름

### 간접 영향
- 함수 이름이 스택 트레이스에서 변수명 기반으로 표시될 수 있음
- `const` 기반 선언으로 바뀌면서 선언 이전 호출이 있으면 문제가 되지만, 현재 검색 범위에서는 해당 패턴이 확인되지 않음
- 오토임포트 기반 Nuxt 환경이므로 import 추가 없이 선언 방식만 변경해야 함

## 실행 판단
### 구현 가능성
- 가능

### 예상 기술 난제
- 선언식 치환 범위가 넓어 누락 없이 처리해야 함
- exported function을 exported const로 바꿀 때 타입 추론과 기존 사용처 호환성을 유지해야 함

### 가정 / 불확실성
- `context7 MCP`는 현재 세션에서 제공되지 않아 지침상 선행 검색을 수행하지 못함
- `src/app` 전체에서 `function` 키워드 검색 기준으로 대상 범위를 선정했으며, 클래스 메서드나 객체 메서드 축약형은 이번 요청 범위에 포함되지 않음
