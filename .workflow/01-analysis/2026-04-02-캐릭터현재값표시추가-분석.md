# 분석 (Analysis)

## 요약
- 요청 내용: 캐릭터 에디터에 현재 레벨, 현재 자금을 표시한다.
- 범위: `CharacterEditorForm.vue`
- 구현 가능성: 가능

## 요구사항 요약
- 현재 레벨은 캐릭터 기준 실제 누적 레벨을 보여 준다.
- 현재 자금은 `currency-transactions`의 캐릭터별 내역 합산값에 세션 보상 골드를 더한 결과를 보여 준다.

## 현재 상태
### 관련 파일
- `src/app/components/characters/CharacterEditorForm.vue`
  - 현재는 시작 레벨과 시작 자금만 입력 가능하다.
- `src/app/composables/currency-transactions/useGetCharacterCurrencyTransactionList.ts`
  - 캐릭터별 통화 거래 목록을 조회한다.
- `src/app/stores/currency.store.ts`
  - 캐릭터 통화 거래 목록 상태를 보유한다.
- `src/server/api/characters/[id].get.ts`
  - 캐릭터 상세 응답에 `currentLevel`, `sessions` 정보를 포함한다.
- `src/server/types/dto/session.dto.ts`
  - 세션의 `rewardGold` 필드를 제공한다.

### 현재 로직 분석
- 수정 화면은 이미 캐릭터 상세를 읽고 있다.
- 통화 거래 목록도 별도 컴포저블로 조회 가능하다.
- 현재 자금은 아직 화면에 표시되지 않는다.

## 영향도 평가
### 직접 영향
- `src/app/components/characters/CharacterEditorForm.vue`

### 간접 영향
- 수정 화면 진입 시 통화 거래 목록 조회가 추가된다.

## 실행 판단
### 구현 가능성
- 가능

### 예상 기술 난제
- 현재 자금 계산 시 거래 합산과 세션 보상 골드 합산을 한 표현으로 정리해야 한다.

### 가정 / 불확실성
- 현재 자금 표시는 화폐 단위별 문자열로 보여 주고, 세션 보상 골드는 GP에 더한다.
- 생성 화면은 현재값이 없으므로 `-`로 표시한다.
