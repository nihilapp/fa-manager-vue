## 사용 규칙
- 활성 작업은 항상 현재 진행 중인 항목만 유지한다.
- 큰 작업은 Plan, Design, Do, Check, Act 흐름에 맞춰 나눈다.
- 완료한 작업은 완료 섹션으로 이동하거나 체크 상태를 갱신한다.
- 우선순위와 의존성이 바뀌면 즉시 반영한다.
